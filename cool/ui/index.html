❄️ Cool UI Update

A slick dark‑mode bookmarklet that upgrades the Cool start page UI.

What it does

  • Set a custom background image (saved locally)
  • Adds a Change background menu option
  • Smooth tile hover opacity
  • Removes greeting headers
  • Hides the widgets button
  • Auto‑minimizes the sidebar
  • Fully transparent top header (logo removed)

Install

  1. Drag the button below to your bookmarks bar
  2. Open the Cool start page
  3. Click the bookmarklet
  4. Press Ctrl / Cmd + Shift + B to change the background
